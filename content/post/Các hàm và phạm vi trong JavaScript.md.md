# **Các hàm và phạm vi trong JavaScript**

# **1. Định nghĩa hàm**
function greet(name) {

    return `Hello, ${name}!`;

}

# **2. Hàm mũi tên**
const square = (x) => x * x;

# **3. Phạm vi biến**
Local Scope: Chỉ hoạt động trong khối lệnh.

Global Scope: Có thể truy cập ở mọi nơi.

# **4. Kết luận**
Hàm và phạm vi giúp tổ chức mã nguồn hiệu quả và tránh xung đột biến.
